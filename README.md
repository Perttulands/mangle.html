## Usage

```bash

git clone https://github.com/ArcadeLabsInc/mangle.html.git

# Open it
open index.html
# or just drag index.html into your browser
```

That's it. No `npm install`. No build step. Just HTML.

## Why?

Because sometimes you just want to visually tweak an HTML file without spinning up a whole dev environment.
